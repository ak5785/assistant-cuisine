# 🍽️ Assistant Cuisine – Anti-Gaspi

Assistant interactif pour planifier tes menus, limiter le gaspillage alimentaire et gagner du temps grâce à l'IA.

## 🚀 Fonctionnalités

- 📸 Analyse d'ingrédients à partir de photos via l'IA
- 🧠 Génération automatique de menus avec GPT
- 🧾 Export des menus en PDF ou vers Notion
- 📊 Interface simple et intuitive avec Streamlit

## 🖥️ Démo rapide

Lancer l'application :
```bash
streamlit run app.py
